# ❤️ Insta Like Button  

A simple Instagram-style **double-tap like button** project made with **HTML, CSS, and JavaScript** 🎨💻✨  

---

## 📸 Demo  
When you **double tap** the image ➡️ A **heart animation ❤️** appears just like Instagram!
